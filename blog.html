What is DOM and what is the purpose of DOM?

DOM means document object model.
DOM is a programming interface for HTML documents. It's creates the HTML elements like an objects. When a HTML elements gets an objects then it will easy to handle to update the content, structure, and style of a document.

I would select HTML elements by using DOM in 5 ways.

2. getElementsByClassName()
3. getElementById()
4. querySelector()
5. querySelectorAll()

What is event bubble?

Event bubble is a DOM event propagation.
Where the event works on the innermost target element and then continuously works on the parents node.
